# GAME #

Game development using pygame (python library)  
P1/LP1 homework

## Installation

### Required
* [pip3](https://pip.pypa.io/en/stable/installing/)
* [pygame](https://www.pygame.org/)

### Installing pygame with pip:
```
pip install pygame

pip3 install pygame
```

## Run

#### Windows:
```
run.py
```

#### Linux:
```
python3 run.py
```
